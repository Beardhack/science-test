# Concept Sets

This demonstration uses synthetic concept labels generated from `config/research_question.yaml`.

For a real-data analysis, replace or add version-controlled concept-set exports here:

- exposure concept set for the study exposure drug class
- comparator concept set for the active comparator drug class
- outcome concept set and position/source restrictions
- type 2 diabetes eligibility concept set
- baseline covariate concept sets where investigator-specified definitions are needed
- negative-control outcome or exposure concept sets after clinical review

Each concept set should include a source vocabulary, code, description, include/exclude flag,
valid date/version metadata when available, and reviewer sign-off.
